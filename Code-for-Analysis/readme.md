## Description of each folder
- 5_or_more_tissues: The main directory where we trained, finetuned, and analyzed our BRONTE model.
- 9_or_more_tissues: The directory where we trained and evaluated our model on GTEx subjects with 9 or 10 tissues.
- all_10_tissues: The directory where we trained and evaluated our model on GTEx subjects all 10 tissues.
- WGCNA: The directory where we analyzed the results from WGCNA. We ran WGCNA in R and the code can be found in R folder.
- MR+coloc: The directory where we analyzed the results from MR and coloc. We ran MR and coloc in R and the code can be found in R folder.
- eQTL-analysis: The directory where we analyzed the results from eQTL-analysis. We ran eQTL analysis in R and the code can be found in R folder.
